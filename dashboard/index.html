🚀 Деплой системы

Система уже задеплоена и работает. Для обновления — просто пушьте в git:

# Обновить и задеплоить (авто-деплой через git hook)
git add . && git commit -m "update" && git push

# Или задеплоить напрямую через Netlify CLI
netlify deploy --prod --dir=.

Сайт живой: javohirbahtiyorov2001-code.github.io/b2b-toshkent
